LÄR DIG OM TEKNIKPROGRAMMET

PROGRAMMERING

På teknikprogrammet får du lära dig grunderna inom programmering...och hur man kan använda det för att lösa problem och skapa egna digitala lösningar. Man börjar ofta med att förstå hur programmering fungerar – till exempel hur man skriver instruktioner som datorn kan följa, hur man använder variabler för att spara information, och hur man kan använda villkor och loopar för att styra vad som händer i programmet.

MATTE & FYSIK

På teknikprogrammet läser man mycket matematik och fysik eftersom de är grunden till all teknik....I matematiken lär man sig mer om algebra, funktioner, geometri och trigonometri, och senare även om derivata och integraler. Det handlar om att kunna räkna och förstå hur man kan använda matematiken för att lösa tekniska problem. I fysiken lär man sig om rörelse, kraft, energi, elektricitet och magnetism. Man får också göra laborationer där man testar teorin i praktiken.

PRODUKTION

På teknikprogrammet får man inom produktion lära sig hur saker tillverkas...från idé till färdig produkt. Man får förståelse för hur man planerar, designar och förbättrar tillverkningsprocesser. Man lär sig om olika material, hur de bearbetas och vilka metoder som används i industrin, som till exempel CNC-bearbetning, 3D-printing och automation. Det ingår också att förstå produktionsekonomi, hållbarhet och hur man kan effektivisera tillverkningen utan att slösa på resurser. Ofta får man arbeta praktiskt i projekt där man planerar och tillverkar en egen produkt, kombinerat med digitala verktyg som CAD för att rita och designa.
